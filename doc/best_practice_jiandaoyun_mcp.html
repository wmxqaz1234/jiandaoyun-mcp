快速开始

环境准备

在使用简道云 MCP 服务器之前，需要完成以下环境准备工作。

1.1 获取 API 密钥

  • 登录简道云平台（https://www.jiandaoyun.com/）
  • 进入「管理后台」→「开放平台」
  • 创建或管理您的 API 密钥

1.2 安装 Node.js 环境

需要 Node.js 18.0 或更高版本。

提示

使用 nvm 可以轻松管理多个 Node.js 版本。

1.3 安装依赖

bash 
npm install

1.4 配置环境变量

创建 .env 文件：

env 
JIANDAOYUN_API_KEY=your_api_key_here
PORT=3000
LOG_LEVEL=info

1.5 启动服务器

模式 适用场景 命令
STDIO 模式 本地开发、调试 npm run start
HTTP 模式 生产环境、远程访问 npm run start:http

typescript

  formId: 'form_id',

常见问题与解决方案

问题 1：无法连接 API

解决方案
  • 检查 API 密钥
  • 验证网络连接
  • 检查防火墙设置

问题 2：字段匹配失败

解决方案
  • 检查字段名称精确匹配
  • 使用 getFormFields 获取字段列表
  • 使用字段 ID 进行精确匹配

问题 3：批量操作部分失败

解决方案
  • 使用事务机制
  • 实现重试机制
  • 记录失败条目

问题 4：查询大量数据超时

解决方案
  • 使用分页查询
  • 只请求需要字段
  • 使用过滤器缩小范围
